VS CODE 다운로드 링크

Visual Studio Code

1. 위의 Visual Studio Code 공식 홈페이지로 접속한다.

2. 윈도우용 Visual Studio Code를 다운로드 한다.

3. 다운로드가 완료되면 VSCode.exe를 실행한다

VS CODE 한국어 설치방법



1. 좌측 사이드 바에 Extensions 버튼을 클릭한다.

2. 검색창에 korean를 검색한다.

3. Install 버튼을 클릭한 후 다운로드가 완료되면 재실행한다.
